The Report Editor

aMs presents the Report Editor dialog when you click the Report button (triangle and pencil icon) on a record listing. The dialog will let you select the fields, the sort order, the subtotalling and other options like printing the report or exporting the data to a file readible by Excel. The editor creates a template that can be used for different selections of records from the same table.

Complete documentation on the Report Editor can be found here.

Saving and Opening a Report template

Report templates can be saved for later use in a different session and with different records selected. Remember that the template is "based" on a "Master Table" so the name you give it should remind you of that table.

Updated 02/20/19

Exporting to Excel

Besides just previewing and printing, the resulting report can be exported as "Tab Deliminted Text" (.txt) which Excel can read. Remember to do a Save As... 'Excel workbook' in Excel after you finish you editing.

Updated 02/20/19

Saving Report as PDF

When a report is "Previewed" it does this by creating a PDF document which you can rename and save so that it can be shared or archived.

Updated 02/20/19

Emailing with HTML

Emailing attachments makes assumptions about the receipents available applications and also just creates one more step for them to see. By sending the report to a HTML document, it can be embedded in the body of the email.

Updated 02/20/19

Adding Columns

Adding columns is easy, but not obvious. Right-Clicking or Control+Clicking on the mouse will popup a menu with choices to add at the beginning, end, before or after a column. It will add a seemingly random field in the new column that you can double click to use the Edit Formula dialog to choose the field you want as well as double clicking the column's label to make it match.

Updated 02/22/19

Renaming, Save as...

When you close the Report Editor dialog that has a report that was opened from a prior save the option to Save as a new document checkbox becomes availiable, essentially a 'Save As' option. Turning on the checkbox give you a new .4qr file, leaving it off will overwrite the original .4qr with the changes you made this session.
