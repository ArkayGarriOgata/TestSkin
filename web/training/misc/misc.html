Adding Last Glue Date to an Excel report

Demonstrates using Excel's VLOOKUP function to merge product codes' last glue date from a lookup table created from aMs's Shortcuts>SQL... menu command. This same technique can be used any time you wish to merge data from two reports into one, provided they share a common column, in this example ProductCode is used. Video below is a bit fuzzy when seen in fullscreen mode, the high resolusion video can be down loaded from here.

Notes: The Excel formula used in the video is repeated below:
=IF($C2<>"",VLOOKUP($C2,LastGlueDateLookup!$A$8:$B$158,2,FALSE),"") and then fill down, change the $C to the column that has your productcode and the LastGlueDateLookup!$A$8:$B$158 to the range in the worksheet that you've added, excluding the column titles. The =IF( ) can be omitted if there are no blank rows in your data.

Creating a Bill of Lading

Demonstrates the basic clicks needed to ship against a Customer Release creating a Bill of Lading. When you print the BOL, an invoice will be generated, the inventory will be relieved, a FG ship transaction will be created, the release will have its actual date and quantity set and the order line's shipped quantity will be updated.
