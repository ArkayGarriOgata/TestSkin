aMs Training Videos

The screencast videos available on this site are intended to give you a quick start with common skills and concepts found in the Arkay Management System (aMs). Effort has been made to keep the videos under 3 minutes. You may find it helpful to use the browser's View menu to zoom in or to use the full screen control on the video so that more detail may be seen.

aMs is written using a database management system called 4D. Comprehensive documentation on many of the topics found on this site can be found in the 4D Design Reference.

The Query Editor

aMs presents the Query Editor dialog when you select Search from the Shortcuts menu, click the Find (magnifier icon) on a record listing, or when the Query Editor radio button is chosen from the ubiquitous "Find records in the Xxxxx table where" dialog.

The Report Editor

aMs presents the Report Editor dialog when you click the Report button (triangle and pencil icon) on a record listing. The dialog will let you select the fields, the sort order, the subtotalling and other options like printing the report or exporting the data to a file readible by Excel. The editor creates a template that can be used for different selections of records from the same table.

The aMs Data Landscape

aMs has been built with a relational database management system called 4D. The better you understand the model aMs uses to store data the more effective you will become with the query editor and the report builder. In the Overview section the system will be dissected in to the major subsytems: Order Management, Production, and Inventory. Emphasis will be placed on which fields are used to relate tables together and how you can benefit from this knowledge.
